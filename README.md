# Transformer Text Generator 🚀

## Overview
This project implements a custom Transformer-based text generator using PyTorch.

## Features
- Custom Transformer architecture
- Tokenizer integration
- Training + Inference pipeline
- Temperature + Top-K sampling

## Example Output

Input:
to be or not to

Output:
to be or not to be a king in the world

## Tech Stack
- Python
- PyTorch

## How to Run

### Train
python train.py

### Inference
python inference.py
